DEWLA SEAFOOD — Ready-to-deploy static website
--------------------------------------------------

This package contains a professional, responsive, ocean-inspired static website for 'DEWLA SEAFOOD'.
Features:
- Modern responsive layout (desktop, tablet, mobile)
- Ocean-inspired visuals & color palette
- Smooth reveal animations and hover interactions
- Bilingual: English & French with a language toggle button
- Contact form (mocked; no backend) — you can connect it to your server or a service like Formspree/Netlify Forms
- Accessible markup and semantic structure
- Ready to deploy to Netlify, Vercel, GitHub Pages, or any static host

FILES:
- index.html
- style.css
- script.js

HOW TO USE:
1. Unzip the package.
2. Upload the files to your hosting provider (Netlify / Vercel / GitHub Pages).
3. To enable a real contact form, connect the form to your backend or a third-party service and change the form action in index.html.

Enjoy — if you want images/logos replaced with your own assets I can add them and repackage the ZIP.
